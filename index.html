Version 1.0 was built at Hip-Hop Hacks.

FLASH Interface (source code)

Desktop Graphical User Interface (GUI) for the FLASH computational physics codebase. FLASH Interface allows users to manage runtime parameters in simulations performed with the FLASH code.

FLASH Interface supports full text search of available parameters, provides descriptions of each parameter's role and function, allows for the filtering of parameters based on categories, performs input validation, and maintains all comments and non-parameter information already present in existing files.

FLASH Interface is created with Electron and works on Mac OSX, Windows, and most Linux distros.

Open Macro (source code)

Web application that allows economists in developing countries to run complex simulations for free.

I wrote a system to manage reading and writing thousands configuration files asynchronously and implemented visualizations for simulations using C3 (D3 extension), including a feature to overlay disparate datasets on the same graph.

Open Macro won the $10,000 first place prize at the 2015 University of Chicago App Challenge and is currently being used by the International Monetary Fund.

Team consisted of Joshua Day, Will Clark, Alex Hays, and me. Developed with UChicago Webservices at the 2015 University of Chicago App Challenge.
