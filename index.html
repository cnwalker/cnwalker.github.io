Open Macro

Web application that allows economists in developing countries to run complex simulations for free.

I created an asynchronous file reading and writing system for managing thousands of files with Node.js, allowing the user to easily run economic models with Dynare/Octave.

I also implemented graphing for simulations using C3 (D3 extension), including feature to overlay disparate datasets on the same graph.

Team consisted of Joshua Day, Will Clark, Alex Hays, and me. Developed with UChicago Webservices at the 2015 University of Chicago App Challenge.
